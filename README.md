# reduxdemo
Redux Demo with ES5 Syntax

A Simple Redux Demonstration for understanding the concept of
 - Store
 - Actions
 - Reducers
 - Async Actions

Run
 - node index.js
 - node asyncActions.js
 
JSON Data
 - JSON data is fetched from "https://jsonplaceholder.typicode.com/users" and user.id is fetched.
